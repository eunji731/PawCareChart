# 멍케어차트 One-Pager(PawCare Chart)

## 한 줄 소개

**반려견의 병원 방문 기록, 진료비, 영수증/사진을 한 곳에서 관리하는 개인용 웹앱**

## 만들게 된 계기

반려견을 키우다 보면 병원 기록, 진료 내용, 비용, 영수증이 여러 곳에 흩어져 관리되기 쉽다.

멍케어차트는 이런 정보를 한곳에 모아 **쉽게 기록하고 다시 찾을 수 있도록** 하기 위해 기획했다.

## 목적

- 반려견 병원/진료 기록을 체계적으로 관리
- 진료비, 약값, 검사비 등 비용 내역 정리
- 영수증, 처방전, 사진까지 함께 보관
- 월별 병원비를 한눈에 확인

## 서비스 설명

멍케어차트는 로그인한 사용자가 반려견 정보를 등록하고,

병원 기록과 비용 기록을 작성·조회·수정할 수 있는 서비스다.

복잡한 커뮤니티 기능 없이, **진료 기록과 병원비 관리**에 집중한 가벼운 웹앱이다.

## 주요 기능

- 회원가입 / 로그인
- 반려견 정보 등록
- 병원 기록 등록 / 조회 / 수정 / 삭제
- 비용 기록 등록 / 조회 / 수정 / 삭제
- 영수증·처방전·사진 업로드
- 월별 비용 조회

## 메뉴 구성

- **홈**: 최근 기록, 이번 달 총 병원비
- **반려견 정보**: 기본 정보, 대표사진
- **병원 기록**: 목록, 상세, 등록/수정/삭제
- **비용 기록**: 목록, 상세, 등록/수정/삭제
- **마이페이지**: 내 정보, 비밀번호 변경

## 사용 스택

- **Frontend**: React, Vite, React Router, Axios, tailWind.css, TS, VS
- **Backend**: Spring Boot, Spring Security, JWT, JPA/MyBatis, intelliJ
- **DB / Infra**: supaBase-PostgreSQL, SSL(HTTPS), 프론트/백 분리 배포, dbever

## 사용 세부 기술

- Spring Security + JWT 인증/인가
- REST API 기반 프론트/백엔드 분리
- 파일 업로드(Multipart) 처리
- 사용자별 데이터 분리 저장
- 월별 비용 집계
- HTTPS 적용

## 기대효과

- 반려견 병원 기록과 비용을 체계적으로 관리할 수 있다
- 영수증/사진까지 함께 보관해 활용성이 높다
- 병원비 지출 흐름을 쉽게 파악할 수 있다
- 인증, 파일업로드, API, SSL, 배포를 포함한 실서비스형 프로젝트 경험이 가능하다
